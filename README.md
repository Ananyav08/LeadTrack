# 🚀 LeadTrack

LeadTrack is a modern Lead Management System designed to help businesses efficiently capture, organize, and manage customer leads. It provides an intuitive dashboard for tracking leads, monitoring their status, and streamlining the sales process through a clean and responsive user interface.

🌐 **Live Demo:** https://lead-track-self.vercel.app/

---

## 📌 Features

- 🔐 User Authentication
- 📊 Interactive Dashboard
- ➕ Add New Leads
- ✏️ Edit Existing Leads
- ❌ Delete Leads
- 🔍 Search & Filter Leads
- 📈 Lead Status Tracking
- 📱 Fully Responsive Design
- ⚡ Fast and User-Friendly Interface

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Ananyav08/LeadTrack.git
```

### 2. Navigate to the Project

```bash
cd LeadTrack
```

### 3. Install Dependencies

For frontend

```bash
cd client
npm install
```

For backend

```bash
cd server
npm install
```

### 4. Configure Environment Variables

Create a `.env` file inside the server folder.

Example:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 5. Run the Application

Backend

```bash
npm run dev
```

Frontend

```bash
npm start
```
